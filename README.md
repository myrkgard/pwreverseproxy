# Reverse Proxy for Personal Website

Nginx reverse proxy

- myrkgard.com:80, www.myrkgard.com:80 -> pwwebsite:80
- dashboard.myrkgard.com:80 -> pwdashboard:80
- api.myrkgard.com:80 -> pwapi:8080
